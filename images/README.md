# Sporting Edge — Website Images

Ready-to-use, web-optimised photos named to match the site's image slots.
Drop this whole folder's contents into your repo's **`images/`** folder
(overwriting the navy placeholders). Filenames match exactly what each page
expects, so nothing in the code needs to change.

Each photo was resized to ~1700px wide and compressed for fast loading.
